Definition of HTML5 Semantic Elements

Semantic elements are those html tags that clearly describe their meaning. Examples of semantic elements are tags, such as, <table>, and <time>.
Non semantic elements that were used in the html4, and earlier versions did not describe its content well. Tags such as, <span>, and <div>.

Below are some of the benefits of semantic elements:

  1. They are supported in all modern browsers, such as, Chrome, Edge, and Firefox
  2. They enable technologies to read and interpret the webpage
  3. They enable searchability by other computers
  4. They make it easy to understand the tags by non-native English speakers
  5. Other developers can better understand the website structure

Some of the main semantic tags for large document outlines are as follows:

  • Header (document header)
  • nav (navigation bar)
  • aside (a sidebar)
  • section (grouping of contents)
  • footer (document footer)
